# Security Policy

## Supported Versions

Only the latest state of the profile `main` branch is supported for security patches.

| Version | Supported |
| ------- | --------- |
| main    | ✅        |
| Older   | ❌        |

## Reporting a Vulnerability

If you discover a vulnerability or security risk regarding this repository (e.g. exposed personal email details, API keys, or unsafe dynamic widget injection points):

1. Do NOT open a public issue.
2. Email me directly at `hi@abidalidev.com`.
3. Provide details on the vulnerability and steps or concepts needed to reproduce.
4. I will investigate and address the issue as soon as possible.

Thank you for helping keep this profile secure!
