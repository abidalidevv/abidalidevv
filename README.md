>
> Automates browser-based text-to-speech workflows with intelligent voice selection, recording automation, multi-language support, and robust state management.

> **Chrome Extension | HTML, JavaScript**
>
> Adds hands-free voice input to ChatGPT with speech recognition, a draggable microphone overlay, language selection, and optional auto-send.

> **Developer Utilities | JS, TS, PHP, Python, SQL, CSS**

> A reference repository containing production-ready utility functions, helper classes, and reusable components for modern web development.

> **Automation Bot | Python**
>
> Windows automation script that controls LDPlayer emulator instances, manages VPN connections, and automates CapCut editing and template publishing processes.
